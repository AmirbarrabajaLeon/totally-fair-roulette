## Features

| # | Idea | Why |
|---|------|-----|
| 1 | Remove winner after spin | Prevents the same person winning twice |
| 2 | Spin history log | See who won previously |
| 3 | Weight adjustment per participant | Subtly bias probabilities instead of a single rigged winner |
| 4 | Save / load via LocalStorage | Names and settings persist across refreshes |
| 5 | Keyboard shortcut (Spacebar = SPIN) | Feels natural, faster |
| 6 | "Fair mode" toggle | Visibly disables rigging to prove it is random |
| 7 | Custom color themes | Pick different palettes for the wheel |
| 8 | Import from comma-separated list | Alternative to one-per-line input |
| 9 | Export winners to text / CSV | Useful for keeping records |
| 10 | Spin speed slider | Let users control how fast it spins |
| 11 | Undo last spin | Recover from accidental spins |
